Bienvenue!

Vous vous trouvez sur un site dont le thème est le langage Brainf***. Ce site va vous présenter et expliquer ce langage dit exotique.

Sur ce site, vous trouverez en-tête le titre du site. A votre gauche, vous trouverez le menu pour savoir sur quelle page vous vous trouvez ainsi que changer de page avec la possibilité de changer de style. Au centre, vous trouverez le corps du site avec son contenu. Au pied de page, vous trouverez un formulaire pour laisser un avis sur le site ainsi que ses auteurs.

Vous vous retrouvez sur la page d'acceuil du site.
Ordre de navigation recommandé:

Pour une expérience optimale, nous vous recommandons de commencer par la page Explication du langage dans lequel nous vous présentons et expliquons le langage, c'est-à-dire comment il fonctionne et comment programmer avec.
Enfin, vous pouvez visitez la page des Sources utilisées pour connaître les sources dans notre travail de recherche dans le langage, mais aussi pour la création du site.

J'espère que vous apprécirez parcourir notre site Web sur le langage exotique Brainf***.
